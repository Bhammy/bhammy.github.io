My Projects

I like to code! These projects are just a small selection of my interests.

Shelf

A full stack single-page web application inspired by Amazon's Goodreads. Designed with responsive user interactivity in-mind, it ensures minimum user navigation to rate, review and create collections of books. A RESTful Rails backend is integrated with React-Redux frontend; a custom auth pattern authenticates user info using BCrypt.

Merchant of Mercury

An arcade-style space shooter designed with HTML5 Canvas & JavaScript. Coded with OOP in mind, every game object keeps track of its internal state to make it easy to keep building out the game itself.
